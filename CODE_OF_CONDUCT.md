# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1, as its code of conduct.

The full text is available at the link above. By participating in this project — as a contributor, reviewer, or community member — you agree to abide by its terms.

## Enforcement

Report code-of-conduct issues privately to the maintainers via GitHub Security Advisories (for sensitive matters) or by opening a confidential issue to the maintainer team.

Maintainers will review and respond within 7 days.
